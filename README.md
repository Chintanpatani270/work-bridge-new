# WorkBridge | Premium Digital Labor Marketplace

**WorkBridge** is a modern, high-conversion web platform designed to connect verified local professionals with customers for home, office, and construction services. It features a seamless user experience for both service seekers and providers.

## üöÄ Features

### For Customers
*   **Smart Search & Filtering:** Easily find services by category (Home, Office, Construction, etc.) or search bar.
*   **Verified professionals:** Access background-checked workers with transparent ratings.
*   **Instant Booking:** Simple modal-based booking flow.
*   **Responsive UI:** Optimized for desktop and mobile with a sleek, glassmorphic design.

### For Workers (New!)
*   **Streamlined Onboarding:** A friction-free 3-step registration process.
*   **Identity Verification:** Secure ID upload simulation (Aadhaar/Voter ID).
*   **Earnings Calculator:** Visual breakdown of potential income.
*   **No Commission:** Direct payment model highlighted.
